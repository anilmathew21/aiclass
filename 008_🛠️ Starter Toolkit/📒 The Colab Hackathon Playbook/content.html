Use Colab's built-in "Secrets" manager to keep them safe.How to do it:Click the key icon (🔑) in the left sidebar.Click "+ Add a new secret".Give your secret a Name (e.g., OPENAI_API_KEY).Paste your key into the Value field.Toggle the switch to give your notebook access.Use this simple code to access it:
from google.colab import userdata
my_api_key = userdata.get('OPENAI_API_KEY')
2. 🛠️ Installing Your Tools in a SnapInstall any Python library you need with a single command. It's best practice to put all your installations in the very first code cell.
# In your first code cell:
!pip install gradio openai llama-index "unstructured[pdf]"
3. ⚡ Unlocking Super Speeds with a GPUFor AI tasks, a GPU is your best friend. It can make your project run up to 10x faster!How to enable it:Go to the menu: Runtime → Change runtime type.From the "Hardware accelerator" dropdown, select T4 GPU.Click Save. The notebook will restart with its new superpowers!4. 📁 Working with Files (Without Losing Them!)Need to upload a document or dataset? Here are your two best options:Temporary Upload: Click the folder icon (📁) on the left and drag your files in.
⚠️ Warning: These files will be deleted when your Colab session ends! This is best for quick tests.
Google Drive (Recommended): For files you need to keep, connect your Google Drive.
from google.colab import drive
drive.mount('/content/drive')
