This exercise aims to use the filter, map, and sort functions on a FIFA players dataset to get their stats and generate an average of all the stats( to find out who is the best all-rounder)Instructions:Take a look at the dataset and identify what values you need to extract.Filter out everything except the integer values from the 1st data point (i.e., data[0]). Use the int_val datatype, which we got from the read_data() to check if it is an integer. Store the results in a list, filter_int.Next, filter out all integers with more than 2 digits from filter_int. Do this by defining a function, count_digit(), that takes a number as input, returning True if the number is a 1-digit or a 2-digit number, and False otherwise.Calculate the average of all these 1 and 2-digit integers, giving you an all-around statistics average for player 1 (i.e., data[0]).Now we will find average stats for all players. For this:Define a function value_cal() that takes elements of the dataset as input and returns the mean of 1- and 2-digit their integer values.Implement this using the filter() function and the count_digit() function we built earlier.Use value_cal() function to calculate average values for all entries in the data list using the map() function.Sort your data list in increasing order of all-around potential (i.e., average of 1 and 2-digit integers), using the sorted() function, and report the best and worst player according to the all-around average values.Using a larger dataset with 10,000 entries, perform the same operations and report the worst and best players according to the all-around average values.Hints:
