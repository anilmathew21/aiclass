Your mission is to design and build a multi-agent system where AIs work together as a software development team to write, test, and review code. This project is all about showcasing the power of specialized agents collaborating to complete a complex task.During the hackathon, your team will build a prototype that brings this AI coding team to life. Your system will:1. Accept a Programming Task:Your app will start with a simple user interface where a user can enter a programming request, like "Write a function that returns the nth Fibonacci number."2. The "Coder Agent" Writes the Code:The first agent in your pipeline will act as the developer. It will take the user's request and generate a clean Python function to solve the task, complete with a proper docstring.3. The "Tester Agent" Creates the Unit Tests:Next, a second, specialized "Tester Agent" will analyze the original request and the Coder's code. Its job is to dynamically write a comprehensive set of unit tests, making sure to cover important edge cases (like empty lists or zero values).4. The "Reviewer Agent" Analyzes the Work:A final "Reviewer Agent" will act as the senior developer on the team. It will perform a full code review by:Running the tests written by the Tester Agent against the Coder Agent's code.Checking for quality using a linter (like Ruff or flake8) to find style issues or potential bugs.Providing clear, constructive feedback based on the results.5. Present the Results in a Clean UI:The entire workflow will be displayed in a simple user interface built with Gradio or Streamlit. The user should be able to see the generated code, the generated tests, and the final review report all in one place.The Goal: The focus isn't on building a perfect, flawless code assistant. It's about demonstrating the power of dividing tasks between specialized agents. A functioning prototype that clearly shows this collaborative workflow for a few programming tasks will be a massive success!
