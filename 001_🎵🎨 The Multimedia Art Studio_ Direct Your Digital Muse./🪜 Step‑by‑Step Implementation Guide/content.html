2. Define the User's Creative VisionThe heart of your application is the user's prompt. This single, rich description will guide all three of your AI artists.Create the Input: In your UI, use a Textbox for the user to describe their scene.Encourage Rich Prompts: This is key! Guide the user to provide evocative, detailed phrases that set a scene. For example: "a lone astronaut watching a supernova from their cockpit," or "a hidden, bioluminescent forest at midnight."3. Compose the SoundtrackThis is the first creative step: generating the audio that sets the emotional tone.API-Based Music Generation (Recommended):Find an API: Use AI music services like Soundraw, Beatoven.ai, or Suno. These services are designed to take a text prompt and return a high-quality audio file.API Call: Write a Python function that sends your user's prompt to the service's API and saves the resulting MP3 or WAV file.4. Write the NarrativeNext, use a Large Language Model (LLM) to write the story or poem that inhabits your scene.Use a Powerful LLM: Call a model like Gemini, GPT-4o, or Claude 3 via their API.Craft a "Director's" Prompt: Your prompt should instruct the model on what kind of text to write for the scene.Example Prompt:
"You are a master storyteller. Your task is to write a short, evocative piece of narrative prose (around 150 words) that captures the feeling of the following scene. Focus on sensory details and internal thoughts.

Scene:
{user's prompt here}"
5. Design the Visual ArtThis is the step that completes the triptych. Generate a stunning image to be the visual centerpiece of your scene.Use an Image Generation API: Call a model like DALL-E 3 (via the OpenAI API) or Stable Diffusion (via the Stability AI API).Create a Visual Prompt: Often, your user's prompt is already a great start, but you can refine it to get more artistic results.Example Prompt (for DALL-E 3):
"An atmospheric, cinematic digital painting of: {user's prompt here}. The style should be moody and evocative, with a focus on light and shadow."
API Call: Write a function that sends this prompt to the image API and saves the returned image file.6. Present the Final ArtboardBuild an interactive and beautiful interface in your notebook using Gradio.Essential Components:A Textbox for the user's scene description.A "Generate Scene" Button.An Image component to display the stunning visual art.An Audio player component to play the generated soundtrack.A Textbox or Markdown component to display the narrative.7. Iterate and RefineA great artistic tool allows for experimentation."Remix" the Scene: Add a button that allows the user to regenerate one of the three elements (e.g., "New Music," "New Image") while keeping the other two. This encourages creative exploration.Document Your Process: In your notebook's text cells, describe your journey. What prompts created the most cohesive results? What was the biggest challenge in making the three parts feel like they belong together? This narrative is a crucial part of your submission.
