The main aim of this exercise is to get you comfortable with reshaping, broadcasting & performing calculations leveraging these two concepts. The exercise is divided into three parts.InstructionsPart 1 - Getting comfortable with reshape, vstack and hstackCreate a 8x8 matrix and fill it with a checkerboard pattern (1,0) using hstack and vstack.Create a 3x3 matrix with values ranging from 0 to 8 using np.arange and np.reshape.Create a 5x5 array containing values from 0 to 24 using np.arange and np.reshape. Subsequently, swap the first two rows of this array using indexing.Perform reshape on the (3x2) matrix provided by transforming it into (3x1x2).Part 2 - Getting comfortable with broadcastingSubtract column-wise mean of an array from each of the elements i.e perform element wise subtraction. Subtract row-wise mean of an array from each of the elements i.e perform element wise subtraction. NOTE: This is not the same as subtracting column-wise means.Part 3 - Calculating Euclidean Distances between pointsSimilar to what the Grand Book Keeper (now the Grand Astronomer) did to calculate the closest star, you will be performing the same task using NumPy to calculate distances between a set of 10 coordinate points of the form (x,y), using the Euclidean formula provided. In order to do the above task, you need to reshape the matrix accordingly for getting it broadcast compatible. Hints
np.hstack
Stack arrays in sequence horizontally (column wise).
np.vstack
Stack arrays in sequence vertically (row wise).
np.reshape
Gives a new shape to an array without changing its data.
np.sum
Return the sum of the array elements over the given axis.
np.mean
